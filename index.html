My interests in these areas reflect my eagerness to explore the diverse and exciting realms that Computer Science has to offer.

What I know about web design so far

This class has been an incredible journey so far, and I've gained a wealth of knowledge in web design. I've delved into various aspects, primarily focusing on HTML, where I've learned both the basic elements and the importance of semantic ones like header, article, nav, and more. Understanding these elements not only enhances the structure of your CSS but also streamlines it, making it more organized and concise. My coding skills have undeniably improved since I started working with HTML and CSS.

In my journey through programming languages, C++ has emerged as my absolute favorite. While I initially began with JavaScript and explored Java and C++ during my courses at Diablo Valley College and San Joaquin Delta College, C++ has always stood out due to its versatility, simplicity, and speed.

Throughout this class, I've encountered a few challenges. One of them is figuring out how to make a single CSS file work for multiple pages and linking all the CSS elements seamlessly. Additionally, creating a multi-page website presented its own set of hurdles. Despite these challenges, I've found great support through discussions with my classmates. It's been enjoyable and reassuring to share and discuss our issues and struggles. This sense of community has reminded me that I'm not alone in my journey to learn about web design.
